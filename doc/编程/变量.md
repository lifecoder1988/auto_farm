变量

变量是用于存放 1 个值的容器，并且可以自定义变量的名字。
= 运算符用于给变量赋值，把右边的值存入左边的变量名中。

let variable_name = value

运算符左边是变量名（你可以自定义），右边是一个表达式，其结果将被存储到变量中。

声明 1 个名为 a 的变量，并在其中存储值 5：
let a = 5
声明 1 个名为 b 的变量，并在其中存储 canHarvest() 的返回值：
let b = canHarvest()

不要把 = 运算符和 == 运算符搞混了。
== 和 === 是判断两个值是否相等的比较运算符，返回 true 或 false。
= 是赋值运算符，把右边的值存入左边的变量。

给变量赋值后，就可以在代码中使用这个变量来替代它包含的值。

let a = 5
for (let i of range(a)) {
	doAFlip()
}

上面的循环执行了 5 次，因为 a 被设置为了 5。
for 循环中的 i 也是一个变量，它在循环的每次迭代中获取序列中的当前值。（它不一定非要叫 i，你可以给它取任何有效的变量名。）

变量也让你能用 while 循环做同样的事情：

let a = 5
let i = 0
while (i < a) {
	doAFlip()
	i = i + 1
}

这和上面的 for 循环做的是同样的事情。我们只需要手动递增 i。
注意，要递增 i，我们把它设为它自己的值加上 1。根据变量的旧值改变其新值是非常常见的操作。

这些运算符可以用来简化代码：+=, -=, *=, /=, %=

i = i + 1 和 i += 1 是一样的  
a = a / 3 和 a /= 3 是一样的
