你的农场变大了！无人机现在可以移动了。  
调用新的 move() 函数即可让无人机移动。  
move() 接受一个方向参数，你可以使用以下四个常量：

North, East, South, West  
分别表示“向上”、“向右”、“向下”、“向左”。

例如：

move(North)

会让无人机向上移动 1 格。

如果移动超过农场边缘，无人机会从另一边回到起始方向。  
下面的代码会让无人机一直向上移动，撞到边缘时从底部回到上方继续移动，直到程序停止：

while (true) {
	move(North)
}
