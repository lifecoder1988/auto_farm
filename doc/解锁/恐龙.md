蛇

蛇是神秘而灵活的生物，可以养殖以获取蛇鳞。

你现在可以让无人机变成一条蛇。  
切换成蛇的方式如下：

change_character(Characters.Snake)

如果你切换为蛇形态，并且拥有足够的仙人掌，则会自动购买一颗苹果并将其放在角色下方。  
当角色站在苹果上并再次移动时，会吃掉该苹果，同时 **蛇身长度增加 1 节**。  
如果仍有足够的仙人掌，则会在随机位置再生成一颗新的苹果。

如果苹果想生成在已被占据的位置，则无法生成。

蛇的身体会跟随移动，填满角色之前走过的路径。  
如果你试图移动到蛇身上，move() 会失败并返回 false。

移动时：
- 蛇身尾端会自动缩短 1 节，因此你可以移动到尾巴原来的位置。
- 如果蛇身填满整个农场，你将无法继续移动。  
  你可以通过检查 move() 是否失败来判断蛇是否已经长到最大。

蛇形态下，角色不能穿越农场边界。

你可以使用 measure() 获取下一颗苹果的位置（以数组形式返回）：

const [next_x, next_y] = measure()

当你切换成其他角色以卸下蛇形态时，会立即收获整条蛇身。  
你将获得 **蛇身长度平方数量的蛇鳞 Items.Scale**。  
如果蛇身长度是 n，则获得 n\*\*2 个蛇鳞：

长度 1 => 1  
长度 2 => 4  
长度 3 => 9  
长度 4 => 16  
长度 16 => 256  
长度 100 => 10000  

蛇形态非常笨重，因此 move() 默认需要 **400 ticks**（普通移动为 200 ticks）。  
但每吃掉一颗苹果，move() 所需的 ticks 会减少 **3%（向下取整）**：

let ticks = 400
for (let i = 0; i < 100; i++) {
	console.log("吃掉 ", i, " 颗苹果后的 ticks: ", ticks)
	ticks -= Math.floor(ticks * 0.03)
}

你只有 **一个蛇形态**，因此同一时间只能有一架无人机变成蛇。
