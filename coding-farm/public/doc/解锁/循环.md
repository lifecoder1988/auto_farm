While 循环

你已经解锁了 while 循环和 true、false 这两个值。while 循环会在条件为 true 的情况下一直执行循环体。

while (条件) {
	// 循环体
}

不用担心创建无限循环。游戏的执行延迟机制会防止程序卡死。

面向初学者

也许你已经试过连续写好几个 harvest() 调用：

harvest()
harvest()
harvest()

这样一次运行程序即可收获多次。  
但如果想超过 3 次，或避免重复写代码，使用循环会更好。

循环能让你多次执行相同的代码。

while 循环需要 1 个条件，该条件的结果必须是布尔值（true 或 false）。

当条件为 true 时，循环会执行循环体；当条件变为 false 时，循环停止。

while 循环的形式如下：

while (条件) {
	// 循环体
	// 循环体
	// ...
}

条件 是布尔值，循环体则是在条件满足时需要重复执行的内容。

JavaScript 也有两个常量布尔值：true 和 false。

例如：

while (false) {
	doAFlip()
}

或者：

while (true) {
	doAFlip()
}

第一个循环永远不会运行，第二个则是无限循环。

虽然无限循环通常不是好主意，因为会卡死程序，但在这款游戏中，每次循环之间都有延迟，因此它只会让无人机不停翻转，直到你按下执行按钮手动停止。

注意 {} 中的内容如何缩进。缩进用于分隔代码块。  
按 Tab 可以缩进一行，Shift + Tab 可以取消缩进。

循环会重复执行大括号中的所有缩进行。  
大括号之后的代码会在循环结束后执行。
