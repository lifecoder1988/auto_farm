变量

变量是用于存放 1 个值的容器，并且可以自定义变量的名字。  
= 运算符用于给变量赋值，把右边的值存入左边的变量中。

let variable_name = value

左边是变量名（可以自定义），右边是表达式，其结果会存储到变量中。

声明一个名为 a 的变量，并存储值 5：
let a = 5

声明一个名为 b 的变量，并存储 canHarvest() 的返回值：
let b = canHarvest()

不要把 = 和 == 搞混了：
- = 是赋值，将值存到变量中  
- ==（以及更严格的 ===）是比较两个值是否相等，返回 true 或 false

给变量赋值后，就可以在代码中使用这个变量替代对应的值。

let a = 5
for (let i of range(a)) {
	doAFlip()
}

上面的循环执行了 5 次，因为 a 的值是 5。
for 循环中的 i 也是一个变量，它在每次迭代中会自动获得序列的当前值。（名字不必须是 i，你可以用任意合法名称。）

变量也能让你用 while 循环做相同的事情：

let a = 5
let i = 0
while (i < a) {
	doAFlip()
	i = i + 1
}

这与上面的 for 循环效果相同。我们只需要手动递增 i。

递增变量是非常常见的操作，以下运算符可以简化写法：+=, -=, *=, /=, %=

i = i + 1 与 i += 1 等价  
a = a / 3 与 a /= 3 等价
