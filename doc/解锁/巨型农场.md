巨型农场

这个极其强大的解锁项让你能够使用多架无人机。

和以前一样，一开始只有一架无人机。额外的无人机必须先被生成，且程序终止后便会消失。
每架无人机运行的都是自己独立的程序。  
可以使用 spawn_drone(function) 来生成新无人机。

function drone_function() {
	move(North)
	doAFlip()
}

spawn_drone(drone_function)

以上代码会在运行 spawn_drone(function) 的无人机所在位置生成一架新的无人机。  
新无人机随后开始执行指定的函数，并在完成后自动消失。

无人机之间不会发生碰撞。

使用 max_drones() 获取可同时存在的无人机数量上限。  
使用 num_drones() 获取当前农场上的无人机数量。

示例：


function harvest_column() {
	for (let _ of range(get_world_size())) {
		harvest()
		move(North)
	}
}

while (true) {
	if (spawn_drone(harvest_column)) {
		move(East)
	}
}

这段代码会让你的第一架无人机水平移动，同时不断生成更多无人机。
新无人机会垂直移动并收获路径上的资源。

如果生成的无人机数量已达上限，spawn_drone() 会返回 null。

以下示例展示如何为每个无人机传入不同方向：

for (const dir of [North, East, South, West]) {
	function task() {
		move(dir)
		doAFlip()
	}
	spawn_drone(task)
}

所有无人机都一样

没有所谓的“主”无人机。  
所有无人机都可以生成其他无人机，并全部计入无人机数量上限。  
无人机在终止时都会消失。  

如果第一架无人机提前完成程序，其它无人机仍然会继续运行，并成为展示代码执行高亮的无人机。

所有无人机都可以触发断点。  
当某架无人机触发断点时，代码高亮会切换到它身上。
