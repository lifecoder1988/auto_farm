运算符

算术运算符：+, -, *, /, %, **  
比较运算符：==, !=, <=, >=, <, >  
布尔运算符：!, &&, ||

注意：游戏中的所有数字都是浮点数，因此所有算术运算都是浮点运算。  
JavaScript 中没有 // 运算符，可以使用 Math.floor(a / b) 实现“向下取整除法”。

若要使用赋值运算符（=、+=、-= 等），需要先解锁“变量”功能。

介绍

运算符用于对值进行比较、修改和组合。  
算术运算符 (+, -, *, /, %, **) 用于执行数学计算。  
比较运算符 (==, !=, <=, >=, <, >) 用于比较数值或其他类型，结果为 true 或 false。  
逻辑运算符 (!, &&, ||) 用于组合布尔值。

---

算术运算符

+ 和 - 用于加法和减法：

2 + 3 的结果是 5  
3 - 2 的结果是 1  

* 和 / 用于乘法和除法：

2 * 3 的结果是 6  
5 / 2 的结果是 2.5  

//（向下取整除法）可写为：

Math.floor(5 / 2) 的结果是 2  

% 是模（取余）运算符：

4 % 2 的结果是 0  
5 % 2 的结果是 1  
6 % 2 的结果是 0  
2 % 6 的结果是 2  
1.5 % 1 的结果是 0.5  

** 是幂运算符：

2 ** 2 的结果是 4  
(-5) ** 3 的结果是 -125  

---

比较运算符

== 和 != 判断两个值是否相等或不相等（类型会自动转换）。  
推荐使用更严格的 === 和 !==。

2 == 2 的结果是 true  
Entities.Bush != Entities.Bush 的结果是 false  
3 != 3 + 1 的结果是 true  

<=, >=, <, > 用于数值比较：

1 <= 1 的结果是 true  
2 >= 3 的结果是 false  
-2 < -1 的结果是 true  
6 > 6 的结果是 false  

---

逻辑运算符

! 用于取反：

!false 的结果是 true  
!true 的结果是 false  

&& 只有两个值都为 true 时，结果才为 true：

true && true 的结果是 true  
true && false 的结果是 false  
false && false 的结果是 false  

|| 在至少一个值为 true 时结果为 true：

true || true 的结果是 true  
true || false 的结果是 true  
false || false 的结果是 false
