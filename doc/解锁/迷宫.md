迷宫

现在你可以在农场里生成迷宫了！

当实体调用 createMaze(size) 将创建一个树篱迷宫。  
迷宫的大小由 size 决定，会生成一个 size × size 的迷宫。

示例：

createMaze(5)

以上代码会创建一个 5×5 的迷宫。

生成迷宫 **不需要任何花费**。

---

迷宫生成后：

- 你无法穿过迷宫的墙壁。
- 你可以通过：

move(dir) → 成功返回 true，否则返回 false  
canMove(dir) → 不移动的情况下检查是否能走  

来判断是否有墙。


在任何迷宫位置调用 measure() 都会返回宝藏的位置：

const [x, y] = measure()

---

宝藏与奖励

当你站在宝藏上时，调用 harvest()：

harvest()

即可收集宝藏。  
宝藏会给予 **等于迷宫面积（size × size）** 的 Items.Gold。

例如：

5×5 迷宫 → 收获 25 金币  
10×10 迷宫 → 收获 100 金币

一旦收获宝藏，迷宫立即消失。


---

迷宫特性


你需要依靠以下方法探索：

- move() / canMove(dir) 判断是否有墙  
- measure() 获取宝藏位置  
- 自己写路径算法寻找宝藏
