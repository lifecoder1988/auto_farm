集合（JavaScript 中的 Set）

集合在 JavaScript 中由 **Set** 类型实现。Set 只存储一组唯一的值（没有 key → value 的映射），类似 Python 的集合。

Set 是无序的，并且每个元素只能出现一次。

创建一个集合：

const set = new Set([North, East, West])

创建一个空集合：

const set = new Set()

添加元素使用 add：

set.add(elem)

删除元素使用 delete：

set.delete(elem)

判断集合中是否包含某个元素使用 has：

if (set.has(elem)) {
	// ...
}

迭代集合中的所有元素：

for (const elem of set) {
	// ...
}

Set 的特点：
- 元素唯一（重复添加不会改变集合）
- 无序迭代
- 查找速度比数组快（特别是大型集合）
- 不支持通过索引访问（因为它不是列表）

示例：

const visited = new Set()
visited.add("A")
visited.add("B")
visited.add("A") // 不会重复添加

console.log(visited.has("A")) // true

for (const v of visited) {
	console.log(v)
}
