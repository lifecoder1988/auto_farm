速度升级

现在，无人机的执行速度已经翻倍！  
但是无人机的收获速度比草的生长速度还快，这会导致收获不到任何作物。  
为了解决这个问题，你现在解锁了 **if 条件分支** 和 **canHarvest()** 函数。

在收获前检查

到目前为止，你只在条件里使用 true 和 false，这对 if 来说意义不大。

现在你可以调用 **canHarvest()** 来判断当前脚下的植物是否成熟。  
如果可以收获 → 返回 true  
否则 → 返回 false

示例：

if (canHarvest()) {
	// 做点什么
}

canHarvest() 的结果是一个布尔值，因此可以直接作为 if 的判断条件。

调用函数并得到一个结果，这个结果叫做“返回值”。

执行上述 if 语句时发生的过程是：

- 遇到 if
- 调用 canHarvest()
- canHarvest() 做检查
- canHarvest() 返回 true 或 false
- if (true) 或 if (false)
- 只有在结果为 true 时，代码块才会执行

这样我们就可以避免在植物未成熟前就提前收获。

使用 if 让无人机在收获前先检查，确保不会浪费动作。
