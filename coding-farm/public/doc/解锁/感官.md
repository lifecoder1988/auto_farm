无人机现在拥有视觉能力了！

调用 get_pos_x() 和 get_pos_y() 会返回无人机当前的 x 和 y 坐标。  
在起始位置时，两者都会返回 0。  
x 坐标向 East（右）方向每格增加 1，  
y 坐标向 North（上）方向每格增加 1。

调用 num_items(item) 会返回你当前拥有某个物品的数量。  
例如：

num_items(Items.Hay)

会返回你现在拥有的干草数量。

调用 get_entity_type() 和 get_ground_type() 会返回无人机脚下的实体或地块类型。

例如，如果无人机在灌木上方，则执行一次翻转：

if (get_entity_type() === Entities.Bush) {
	doAFlip()
}

JavaScript 中的 null 现在解锁了！  
null 表示“没有值”的特殊值。

例如，一个没有 return 的函数实际会返回 null：

function f() {
	// 无返回值
}
let r = f() // r === null

如果脚下没有实体，get_entity_type() 会返回 null。

---

如果想知道你拥有某个 **科技树项目** 的等级或是否解锁，可以调用 num_unlocked(unlock)。

例如：

num_unlocked(Unlocks.Speed)

这会返回你当前的速度等级。

如果你已经解锁了“感官”科技树：

num_unlocked(Unlocks.Senses)

会返回 1，否则返回 0。

你也可以对物品、实体或地块调用 num_unlocked()：  
- 已解锁 → 返回 1  
- 未解锁 → 返回 0

请注意：

- num_unlocked(Unlocks.Carrots)  
  → 会返回胡萝卜是否被解锁 **或** 胡萝卜的等级（多级科技）

- num_unlocked(Items.Carrot)  
  → **只会**返回 0 或 1  

（其他植物也相同）
