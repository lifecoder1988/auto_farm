运算符

JavaScript 中同样提供了算术运算符、比较运算符与逻辑运算符。  
所有数字都是浮点数，因此所有算术运算符都是浮点运算。

JS 中没有 // 运算符，但可以使用 Math.floor(a / b) 来实现“向下取整的除法”。

介绍

运算符可以用来对值进行比较、修改和组合。  
算术运算符 +, -, *, /, %, ** 用于数字运算。  
比较运算符 ==, !=, <=, >=, <, > 用于比较值，返回 true 或 false。  
逻辑运算符 !, &&, || 用于组合布尔结果。

算术运算符

+ 和 - 用于加法和减法：

2 + 3 的结果是 5  
3 - 2 的结果是 1  

* 和 / 用于乘法和除法：

2 * 3 的结果是 6  
5 / 2 的结果是 2.5  

JS 没有 // 运算符，可用 Math.floor 实现：

Math.floor(5 / 2) 的结果是 2  

% 是模（取余）运算符：

4 % 2 的结果是 0  
5 % 2 的结果是 1  
6 % 2 的结果是 0  
2 % 6 的结果是 2  
1.5 % 1 的结果是 0.5  

** 是幂运算符：

2 ** 2 的结果是 4  
(-5) ** 3 的结果是 -125  

比较运算符

== 和 != 判断两个值是否相等或不相等（需要注意 JS 的类型转换行为，推荐使用 === 和 !==）：

2 == 2 的结果是 true  
3 != 3 + 1 的结果是 true  

=== 和 !== 是更严格的比较方式（不会自动进行类型转换）。

<=, >=, <, > 用于比较数字：

1 <= 1 的结果是 true  
2 >= 3 的结果是 false  
-2 < -1 的结果是 true  
6 > 6 的结果是 false  

逻辑运算符

! 用于取反：

!false 的结果是 true  
!true 的结果是 false  

&& 仅在两个值都为 true 时返回 true：

true && true 的结果是 true  
true && false 的结果是 false  
false && false 的结果是 false  

|| 只要有一个值为 true，就返回 true：

true || true 的结果是 true  
true || false 的结果是 true  
false || false 的结果是 false
