扩张 2

你的农场再次扩张了！现在地块不再是整齐的一行，因此你需要想办法遍历整个方形网格。

如果只使用 while 循环，而你又没有解锁感官或运算符，那么是无法做到这一点的。  
所以现在是时候介绍 **for 循环** 了。

你可以在 <u><link="docs/scripting/for.md">For 循环</link></u> 页面查看更多内容。  
目前你只需要知道，for 循环可以让你固定次数重复执行代码。

示例：执行 5 次翻转

for (const i of range(5)) {
	doAFlip()
}

range(n) 表示一个从 0 到 n-1 的数字序列，包含 n 个元素。  
for 循环会对序列中的每个元素执行一次循环体。  
在上例中，doAFlip() 会运行 5 次。

---

你现在也可以使用 get_world_size()。  
它返回你农场的边长。这样即使农场继续扩张，你写的代码也不会失效。

示例：收获农场第一列的所有格子

for (const i of range(get_world_size())) {
	harvest()
	move(North)
}

无论农场有多大，无人机都会收获第一列的所有地块。
