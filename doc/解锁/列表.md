列表（JavaScript 中的数组）

在 JavaScript 中，列表由 **数组 Array** 实现。数组可以通过一个变量存储多个值。

新数组的创建方式如下：

let list = [2, true, Items.Hay]

这个数组现在包含了 2、true 和 Items.Hay 这些值。
数组也可以是空的：

let empty_list = []

你可以通过索引访问数组中的元素。第一个元素的索引是 0，第二个是 1，第三个是 2……

示例（种植胡萝卜）：

let list = [Entities.Tree, Entities.Carrot, Entities.Pumpkin]
plant(list[1])

你可以使用 for...of 循环迭代数组。下面的例子计算数组中所有元素的和：

let list = [4, 7, 2, 5]
let sum = 0
for (const number of list) {
	sum += number
}
// sum 现在是 18

以下为常用数组方法，可用于添加和删除元素：

list.push(elem) 在数组末尾添加 1 个元素：

let list = [2, 6, 12]
list.push(7)
// list 现在是 [2, 6, 12, 7]

删除某个元素第一次出现的项可以结合 indexOf 与 splice：

let list = [1, 2, 4, 2]
list.splice(list.indexOf(2), 1)
// list 现在是 [1, 4, 2]

list.splice(index, 0, elem) 在给定索引插入 1 个元素：

let list = [Entities.Tree, Items.Hay]
list.splice(1, 0, Items.Wood)
// list 现在是 [Entities.Tree, Items.Wood, Items.Hay]

移除元素可以使用 pop（移除最后一个），移除指定索引则用 splice：

let list = [3, 5, 8, 25]
list.pop()
// list 现在是 [3, 5, 8]
list.splice(1, 1)
// list 现在是 [3, 8]

数组长度可通过 length 属性读取：

let list = [3, 2, 1]
let x = list.length
// x 现在是 3

数组具有引用语义。将一个数组赋给另一个变量时，不会复制数组，而是两个变量共享同一个数组对象。

let a = [1, 2]
let b = a
b.pop()
// a 和 b 现在都是 [1]
